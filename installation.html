Installation¶

Python Package¶

Install package with:

#execute next line being in `src` dir where `setup.py` file is
#after cloning repo:
#https://github.com/lecepe00/mex_simplex.git
pip install .

or in editable mode:

#execute next line being in `src` dir where `setup.py` file is
#after cloning repo:
#https://github.com/lecepe00/mex_simplex
pip install -e .

or with url:

pip install "git+https://github.com/lecepe00/mex_simplex.git#egg=mex&subdirectory=src"
